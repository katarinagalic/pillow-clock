About PillowTime System

Aside from the website portion, the Pillow Alarm Clock also includes the hardware device which consists of:

  • Adafruit Feather M0 Adalogger
  • DYIMore DS3231 AT24C32 IIC High Precision Real Time Clock Module Clock Timer Memory Module For Arduino 
  • Force Sensitive Resistor - Pressure sensors
  • Coin vibration motor 
  • MicroSD Card

The Feather controls the circuit, with the Time Module keeping track of time. Based on the time set through the website, the Feather controls the vibration motors and makes them go off at an appropriate time. The Pressure sensors send data based on whether they sense any pressure applied, thus enabling for tracking sleep data and the snooze aspect of the alarm. MicroSD card stores all of the user's sleep data.
