About PillowTime

Welcome to PillowTime!



PillowTime is a project started by five freshmen at Drexel University, who aimed to revolutionize the sleeping experience. It all started from wanting to prevent waking up our roommates. This website is a way for users to interact with our Arduino module, which represents the core of our project. 

 

How to use:

  • About Tab - brief overview of the website features
  • Add User Tab - allows for adding new users and setting an alarm for the first time
  • Users Tab - shows a list of users linked with this computer.
    • Clicking on  a user leads you to a page similar to the Home Tab, and it allows for setting a new alarm, as well as seeing the sleeptracking data associated with the use
  • Data Entry Tab - allows users to manually enter their sleep data, in case they missed a night of sleep on the pillow tha contains the module

 

Happy sleeping!
